# react-todo


第一个 React 项目，使用 React 制作一个任务清单应用。

[Demo 页面](http://caiyongmin.github.io/react-todo/)

## 使用

```bash
git clone https://github.com/caiyongmin/react-todo.git
```

```bash
npm install
```

```bash
npm run dev
```
